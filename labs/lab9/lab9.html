Inline CSS

Inline CSS allows you to apply style to a single element. For inline style to work you have to add the style attribute to the element you want to "style".

Internal CSS

Internal CSS can be used to add style to only page of your html. For this one to work you have to add a style element in the head section. To show what the rest of the page looks like.

External CSS

External CSS can help you change the style of the entire html webpage you have created. For the CSS to work in every page you have, you have to add a link in the head section of everypage to signify the html that you are using a certain style.
